# Student Course Load & Co-Enrollment Analysis

Este proyecto realiza un análisis exploratorio de datos sobre la carga académica de estudiantes y los patrones de co-inscripción entre materias de una base de datos de inscripciones de materias.

## Objetivos

- Validar la integridad del dataset
- Analizar la distribución de carga académica por estudiante y sección
- Identificar patrones de inscripción por prefijo de materia
- Calcular estadísticas descriptivas de materias por estudiante
- Detectar combinaciones frecuentes de materias (co-inscripción)

## Análisis realizado

- Conteos básicos y verificación de registros únicos
- Histogramas de secciones y carga académica
- Agrupación por prefijos de materia
- Media, mediana y desviación estándar de materias por estudiante
- Generación de pares de materias más comunes entre estudiantes


## Tecnologías

- Python
- Pandas
- Matplotlib
- Google Colab

## Estado

Proyecto de análisis exploratorio enfocado en encontrar patrones de comportamiento académico a partir de datos reales.
